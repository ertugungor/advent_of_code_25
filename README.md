# Advent of Code 2025

Solutions for Advent of Code 2025 in C++.

## Setup

1. Place your puzzle input files (`long.txt`) in the corresponding `puzzles/dayXX/` folders for CMake to locate them during builds and runs.
2. Build the project using CMake.
3. Run the executables from the `build/` directory.

Tested on Ubuntu 20.04 LTS with Clang 18.
